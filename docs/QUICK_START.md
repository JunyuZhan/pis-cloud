# 新功能快速开始指南

## 🎨 图片风格预设 - 3步上手

### 步骤1：创建相册时选择风格
```
管理后台 → 新建相册 → 展开「风格设置」→ 选择风格 → 创建
```

### 步骤2：为现有相册更改风格
```
相册详情 → 设置 → 展开「风格设置」→ 选择新风格 → 确认重新处理 → 保存
```

### 步骤3：查看效果
- 新上传的照片会自动应用风格
- 更改风格后，系统会提示重新处理现有照片

---

## 📥 批量下载 - 2步开启

### 步骤1：开启批量下载权限
```
创建相册时：批量下载设置 → 开启开关
或
现有相册：设置 → 允许批量下载 → 开启 → 保存
```

### 步骤2：访客使用
```
相册页面 → 选择照片 → 点击「下载已选」按钮
```

---

## 💡 重要提示

1. **风格预设**：
   - 默认：无风格
   - 更改风格后需要重新处理照片才能看到效果
   - 13种预设可选（人像5种、风景4种、通用3种）

2. **批量下载**：
   - 默认：关闭（需要管理员明确开启）
   - 需要同时开启「允许下载」和「允许批量下载」
   - 访客只能下载已选中的照片

3. **模板继承**：
   - 如果创建相册时选择了模板，批量下载设置会继承模板配置
